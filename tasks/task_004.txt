# Task ID: 4
# Title: Implement Media Database and Library Management
# Status: done
# Dependencies: 1, 3
# Priority: high
# Description: Create a local database structure using PGLite to store and manage the user's media library, playlists, and application settings.
# Details:
1. Design and implement database schema for media library:
   ```typescript
   // Database schema
   const mediaSchema = `
     CREATE TABLE IF NOT EXISTS media (
       id TEXT PRIMARY KEY,
       title TEXT,
       artist TEXT,
       album TEXT,
       duration INTEGER,
       path TEXT,
       type TEXT,
       provider TEXT,
       thumbnail TEXT,
       added_date INTEGER
     );
     
     CREATE TABLE IF NOT EXISTS playlists (
       id TEXT PRIMARY KEY,
       name TEXT,
       description TEXT,
       created_date INTEGER
     );
     
     CREATE TABLE IF NOT EXISTS playlist_items (
       playlist_id TEXT,
       media_id TEXT,
       position INTEGER,
       PRIMARY KEY (playlist_id, media_id),
       FOREIGN KEY (playlist_id) REFERENCES playlists(id),
       FOREIGN KEY (media_id) REFERENCES media(id)
     );
     
     CREATE TABLE IF NOT EXISTS settings (
       key TEXT PRIMARY KEY,
       value TEXT
     );
   `;
   ```
2. Create data access layer for CRUD operations on media items
3. Implement playlist management functionality
4. Create settings storage and retrieval
5. Implement search and filtering capabilities
6. Create library views (by artist, album, recently added, etc.)
7. Add sorting and organization options
8. Implement database migrations for future updates

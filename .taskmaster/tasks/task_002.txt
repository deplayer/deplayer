# Title: Implement Core Playback Engine
# Status: done
# Dependencies: 1
# Priority: high
# Description: Develop the core media playback functionality that supports both audio and video formats with standard playback controls.
# Details:
1. Create a MediaPlayer component using HTML5 audio/video elements
2. Implement playback controls:
   - Play/Pause toggle
   - Stop functionality
   - Next/Previous track navigation
   - Volume control with mute option
   - Seek bar with current time/duration display
3. Create custom hooks for media playback:
   ```typescript
   function useMediaPlayer(mediaType: 'audio' | 'video') {
     const [isPlaying, setIsPlaying] = useState(false);
     const [currentTime, setCurrentTime] = useState(0);
     const [duration, setDuration] = useState(0);
     const [volume, setVolume] = useState(1);
     const mediaRef = useRef<HTMLAudioElement | HTMLVideoElement>(null);
     
     // Implement play, pause, seek, volume methods
     
     return {
       isPlaying,
       currentTime,
       duration,
       volume,
       mediaRef,
   ```

4. Implement keyboard hotkeys:
   - Space: Play/Pause
   - Arrow right or j: Next track
   - Arrow left or k: Previous track
   - Control + k: Open command bar
5. Create a playlist queue manager to handle track sequencing
